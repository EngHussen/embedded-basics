# Smart Sensor System – Arduino Project

## Overview
This project reads data from an LDR and PIR sensor and controls a relay based on the readings.

## Technologies
- Arduino Uno
- C Programming
- LDR / PIR Sensors
- Relay Module

## Features
- Real–time sensor reading
- Automatic light control
- Serial monitoring
- Safe electrical isolation
